Программа оснащена графическим интерфейсом с 2 текстовыми полями, 1 окном вывода и 8 кнопками.

Перед запуском основной составляющей программы пользователь должен выбрать режим, с которым он хочет работать. Это позволяют кнопки Video, обеспечивающая работу с готовым видеофайлом и Web, включающая работу с видеопотоком. Также возможна интеграция работы программы с Вконтакте, посредством отправки pdf файла с результатом работы. Для этого необходимо ввести Login и Password в соответствующих полях ввода. Далее пользователь должен выбрать соответстующий заданию режим работы. 
Каждому режиму работы соответствует кнопка и с изображением исполняемого действия.


-----------------------------------------------------------------------------
Кнопки:
Video - переход в режим видео и выбор видео, с которым предстоит работа
Web - переход в режим работы Realtime
5 в ряд выстроенных кнопок:
 1.Выборка аномалии падения коробки
	Открывается дополнительное окно, в котором пользователь выделяет зону погрузки(куда кладутся коробки) и зону конвеера(откуда берутся коробки) и программа автоматически отслеживает перемещение коробки между двумя этими зонами.
 2.Выборка аномалии открытия коробки
	Открывается дополнительное окно, в котором посредством QR-кодов программа определяет открыта коробка или нет. В случае обнаружения поступает уведомление. 
 3.Отслеживание просроченных товаров
	Открывается дополнительное поле, в котором посредством QR-кодов программа определяет просрочен товар или нет. В случае обнаружения поступает уведомление. 
 4.Подсчет количества товаров
	Открывается дополнительное поле, в котором посредством QR-кодов программа производит подсчет количества коробок, с выводом результата по ходу выполнения.
 5.Оптимизация укладки коробок в паллету
	Открывается дополнительное окно, в котором пользователь должен ввести в соответствующие разделы физические габариты паллеты и коробки, а программа, на основе реализованного нами алгоритма, предлагает пользователю оптимальный способ укладки.
Exit - выход из программы

Поля ввода:
Login и Password - авторизационные данные для интеграции в Вконтакте(авторизация не работает без корректно введенных данных)

Поля вывода:
Текстовое поле, выводящее на экран состояние/готовность выполнения задания
